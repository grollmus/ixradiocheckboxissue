"jumping" ix-radio and ix-checkbox issue in 2.6.1 (and likely in 2.7.0 too)

1.) Single radio button jumps several vertical pixels when clicked. With or without wrapped in div. Can also be seen in 2.7.0 documentation online link

2.) same jumping behaviour when clicked, but only when wrapped in div (or in tst-ui-radio in our case).

3.) wrapped works only when style display:block is applied.

4.) checkbox jumps not itself, but height is changed a few pixels when clicked (see border below) resulting in controls below jumping up and down
